# fastapi_tasks

**Описание:** простой api для создания тасок 

## Build and run with Docker

1. Build
```bash
docker build . --tag fastapi-tasks
```

2. Run
```bash
docker run -p 8085:8085 fastapi-tasks
```

3. Open http://127.0.0.1:8085/
